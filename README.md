# 📈 台股自動選股機器人

這是一個使用 Python + GitHub Actions 的自動選股系統  
每天會自動分析股票並找出最有潛力的前10名 📊

---

## 🚀 功能

✅ 每天自動執行  
✅ 掃描多檔台股 
✅ 計算技術指標（RSI / MA / MACD）  
✅ 排名出最強10檔股票

---

## 🧠 選股邏輯

- RSI < 30 → 超賣（可能反彈）
- MA5 > MA20 → 上升趨勢
- 收盤價上漲 → 動能
- MACD 黃金交叉 → 多頭訊號

👉 分數越高 = 越有機會上漲

---

## ⏰ 自動執行時間

每天台灣時間 15:00 自動執行
